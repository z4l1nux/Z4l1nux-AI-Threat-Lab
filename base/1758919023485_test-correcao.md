# Sistema de Loja Virtual

## Tecnologias
- Frontend: Vue.js 3
- Backend: Python Django
- Database: PostgreSQL
- Cache: Redis

## Funcionalidades
- Catálogo de produtos
- Carrinho de compras
- Checkout integrado
- Gestão de pedidos

## Segurança
- HTTPS obrigatório
- Autenticação JWT
- Validação CSRF
- Rate limiting
